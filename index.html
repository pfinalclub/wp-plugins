MBTI Personality Test WordPress

A modern MBTI test plugin by PFinalClub for WordPress with responsive design, localization support, form validation, and instant result display.

Key Features

  • 16 questions covering 4 dimensions, auto-navigation after selection
  • "Submit" button only on the last question with completeness validation
  • Multi-language support (English/Chinese), JSON local question bank
  • Secure output and standardized implementation (WP standards compliant)
  • Created by PFinalClub for WordPress community

Quick Start

  1. Upload and activate the WordPress plugin
  2. Create a new page and insert shortcode [mbti_test]
  3. Configure via "MBTI" menu in the WordPress admin dashboard

Compatibility

WordPress 5.0+, PHP 7.4+. Readme marked as Tested up to 6.8. Compatible with latest WordPress versions.
