MBTI Personality Test

A modern MBTI test plugin with responsive design, auto-navigation, validation prompts, and multi-language support.

Key Features

  • Complete MBTI assessment (16 questions covering 4 major personality dimensions)
  • Auto-navigation to next question after selection, reducing user effort
  • "Submit" button only on last question with completeness validation and friendly prompts
  • JSON local question bank with automatic language detection (English/Chinese)
  • Follows WordPress coding standards with secure escaping of all dynamic text

Usage Instructions

  1. Install and activate the plugin
  2. Insert the shortcode [mbti_test] into any page or post
  3. Configure basic settings via the "MBTI" menu in the admin dashboard

Compatibility

WordPress 5.0+, PHP 7.4+. Marked as Tested up to 6.8 in the Readme.
