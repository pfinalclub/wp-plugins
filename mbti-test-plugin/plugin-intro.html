MBTI Personality Test

现代化的 MBTI 测试插件，提供响应式界面、自动跳题、验证提示、多语言支持等能力。

功能特性

  • 完整 MBTI 评估（16 题，覆盖 4 大性格维度）
  • 选择后自动跳到下一题，减少操作成本
  • 仅在最后一题显示“提交”，提交前进行完整性校验并友好提示
  • JSON 本地题库，按站点语言自动加载（中文 / 英文）
  • 遵循 WordPress 规范，所有动态文本安全转义输出

使用说明

  1. 安装并激活插件
  2. 在页面或文章中插入短代码 [mbti_test]
  3. 后台 “MBTI” 菜单可进行基础配置

兼容性

WordPress 5.0+，PHP 7.4+。Readme 标注 Tested up to 6.8。
